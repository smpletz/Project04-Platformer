# Project04-Platformer

## Implementation
Added sound effects to all interactable objects
- Player footsteps
- Player jump
- Enemy death (all enemies)
- "Bounce" noise for jumping enemy
- Footsteps for walking enemies
- "Whoosh" effect for charging enemy
